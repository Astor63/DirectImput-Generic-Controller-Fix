Installation:

1. Install the official Xbox 360 Controller driver.
2. Run ScpDriver.exe, and wait until it finishes to close it
3. Copy X360ce_64 in the same repertory with the [NameOfYourGame].exe 
4. Run X360ce_64 setup as Administrator.
5. Run XOutput and set up your controller mappings identical to X360ce_64

How to use this:

1. Start XOutput
2. Start your game
3. Play with any controller
